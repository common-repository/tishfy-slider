== Description ==

[Demo](https://tishonator.com/plugins/tishfy-slider)

Configure a Responsive Slick jQuery Carousel Slider and Insert it in any Page or Post as a Shortcode. Admin slide fields for pre-title, title, text, image.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the `tishfy-slider` folder to the to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Navigate to Admin Panel -> 'Tishfy Slider' -> fill Slider fields
4. Insert the shortcode [tishfy-slider] to display Slider to any page or post
5. Enjoy!
